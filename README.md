# Install

```bash
  $ git clone https://github.com/tellijo/dotfiles $HOME/.dotfiles
```

```bash
  $ ./install.sh -h

  Script usage: ./install.sh [-a] [-d] [-c] [-i 1] [-y] [-h]

  Options:
    -a          Run all steps
    -d          Debug mode
    -c          Install casks
    -i [step]   Run specific [step]
    -y          Answer yes to all prompt

  Help:
    -h          Show help
```
